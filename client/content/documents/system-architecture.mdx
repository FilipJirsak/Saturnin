## Základní komponenty

### Frontend
- **Framework**: Remix.js
- **Styling**: TailwindCSS, shadcn/ui
- **Stav aplikace**: Remix loaders/actions, místně useState

### Backend
- **Runtime**: Deno
- **API**: REST + částečně GraphQL
- **Databáze**: SurrealDB

## Bezpečnostní model
Systém implementuje následující bezpečnostní mechanismy:
- JWT autentizace
- Role-based access control
- Šifrování citlivých dat

## Integrace s externími systémy
- GitHub API (PROJ-1)
- Jira webhook
- Slack notifikace
